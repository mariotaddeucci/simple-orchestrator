# simple-orchestrator

CLI that can start the `simple-orchestrator-worker` and/or `simple-orchestrator-tui`.

## Install

- Worker: `uv add "simple-orchestrator[worker]"`
- TUI: `uv add "simple-orchestrator[tui]"`
- Everything: `uv add "simple-orchestrator[all]"`

## Run

```bash
uv run simple-orchestrator worker
uv run simple-orchestrator tui
```
